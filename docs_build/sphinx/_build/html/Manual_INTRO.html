Introduction¶

Library for the course Mechatronics System Simulation

Installation¶

Download the toolbox from here and follows the instruction.

Library structure¶

  • DIAL_ODEsystem base class to define the system of Ordinary Differential Equations (ODEs) to be integrated.

  • DIAL_ODEsolver base class to define the solver for the system of ODEs.

  • DIAL_RKexplicit base class to define the explicit Runge-Kutta solver.
