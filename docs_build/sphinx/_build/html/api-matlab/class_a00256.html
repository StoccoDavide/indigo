class RKexplicit : public ODEsolver¶

Class implementing the abstract function step for the explicit Runge-Kutta integration method. The user must simply define the Butcher tableau of the Runge-Kutta method:

\[\begin{split}\begin{array}{c|c} c & A \\ \hline & b \end{array}.\end{split}\]

Subclassed by Collatz, ExplicitEuler, ExplicitMidpoint, Heun2, Heun3, RK3, RK38, RK4, Ralston2, Ralston3, Ralston4, SSPRK3

function step(t_k, x_k, d_t)¶

Compute an integration step using the explicit Runge-Kutta method:

\[\begin{split} \begin{array}{rcl} x_{k+1} &=& x_k + \displaystyle\sum_{j=1}^s b_j K_j \\ K_i &=& h f \left( t_k + c_i \Delta t, x_k + \displaystyle\sum_{j=1}^{i-1} A_{ij} K_j \right), \qquad i = 1, 2, \ldots, s. \end{array} \end{split}\]

The method requires the following inputs:

  • x_k States value at \( k \)-th time step \( \mathbf{x}(t_k) \);

  • t_k Time step \( t_k \);

  • d_t Advancing time step \( \Delta t\).

The return value is the approximation of \( \mathbf{x_{k+1}}(t_{k}+\Delta t) \).

function project(x_tilde, x_dot_tilde, t)¶

Solve the ODEs system through the problem:

\[\begin{split} \textrm{minimize} \quad \dfrac{1}{2}\left( \begin{bmatrix} \mathbf{x} \\ \mathbf{x}' \end{bmatrix} - \begin{bmatrix} \tilde{\mathbf{x}} \\ \tilde{\mathbf{x}}' \end{bmatrix} \right)^2 \quad \textrm{subject to} \quad \mathbf{H}(\mathbf{x}, \mathbf{x}', t) = \mathbf{0} \end{split}\]

given the Lagrangian of the form:

\[\begin{split} \mathcal{L}(\mathbf{x}, \mathbf{x}', \boldsymbol{\lambda}) = \frac{1}{2}\left( \begin{bmatrix} \mathbf{x} \\ \mathbf{x}' \end{bmatrix} - \begin{bmatrix} \widetilde{\mathbf{x}} \\ \widetilde{\mathbf{x}}' \end{bmatrix} \right)^2 + \boldsymbol{\lambda} \cdot \mathbf{H}(\mathbf{x}, \mathbf{x}', t). \end{split}\]

The solution of the problem is obtained by solving the following:

\[\begin{split} \left\{\begin{array}{l} \begin{bmatrix} \mathbf{x} \\ \mathbf{x}' \end{bmatrix} + \mathbf{JH}_\mathbf{x}^T \boldsymbol{\lambda} = \begin{bmatrix} \widetilde{\mathbf{x}} \\ \widetilde{\mathbf{x}}' \end{bmatrix} \\[1em] \mathbf{H}(\mathbf{x}, \mathbf{x}', t) = \mathbf{0} \end{array}\right. \end{split}\]

Solution algorithm

Using the Taylor expansion of the Lagrangian:

\[\begin{split} \mathbf{H}(\mathbf{x}, \mathbf{x}', t) + \mathbf{JH}_\mathbf{x} \begin{bmatrix} \delta\mathbf{x} \\ \delta\mathbf{x}' \end{bmatrix} + \mathcal{O}\left(\left\| \begin{bmatrix} \delta\mathbf{x} \\ \delta\mathbf{x}' \end{bmatrix} \right\|^2\right) = \mathbf{0} \end{split}\]

define the iterative method as:

\[\begin{split} \begin{bmatrix} \mathbf{x}_{k+1} \\ \mathbf{x}_{k+1}' \end{bmatrix} = \begin{bmatrix} \mathbf{x}_{k} \\ \mathbf{x}_{k}' \end{bmatrix} + \begin{bmatrix} \delta\mathbf{x} \\ \delta\mathbf{x}' \end{bmatrix}. \end{split}\]

Notice that \( \left[\delta\mathbf{x}, \delta\mathbf{x}'\right]^T \) is the solution of the linear system:

\[\begin{split} \left[\begin{array}{c|c} \\[-0.75em] \mathbf{I} & \mathbf{JH}_\mathbf{x}^T \\[0.5em] \hline \mathbf{JH}_\mathbf{x} & \mathbf{0} \end{array}\right] \left[\begin{array}{c} \delta\mathbf{x} \\ \delta\mathbf{x}' \\ \hline \boldsymbol{\lambda} \end{array}\right] = \left[\begin{array}{c} \widetilde{\mathbf{x}} - \mathbf{x}_k \\ \widetilde{\mathbf{x}}' - \mathbf{x}'_k \\ \hline -\mathbf{H}(\mathbf{x}_k, \mathbf{x}_k', t_k) \end{array}\right] \end{split}\]

where \( \mathbf{JH}_\mathbf{x} \) is the Jacobian of the invariants/ hidden constraints with respect to the states \( \mathbf{x} \).

function solve(t, x_0, varargin)¶

Solve the system of ODEs and calculate the approximate solution.

Usage:

Solve without the solution projection on invariants/hidden constraints and disabled verbose mode.

sol = obj.solve(t, x_0);

Solve with the solution projection on invariants/hidden constraints and disabled verbose mode.

sol = obj.solve(t, x_0, true);

Solve without the solution projection on invariants/hidden constraints and enabled verbose mode.

sol = obj.solve(t, x_0, false, true);

Plot the first component of the solution.

plot(t, sol(1,:));
Parameters

  • t – Time vector \( \left[ t_0, t_1, \ldots, t_n \right]^T \).

  • x_0 – Initial states value \( \mathbf{x}(t_0) \).

  • project – [optional, default = false] Apply projection to invariants at each step.

  • verbose – [optional, default = \( \mathrm{false} \)] Activate vebose mode.

  • epsilon – [optional, default = \( 1.0\mathrm{e}3 \)] If \( || \mathbf{x} ||_{\infty} > \varepsilon \) the computation is interrupted.

Returns

A matrix \( \left[(\mathrm{size}(\mathbf{x}) + \mathrm{size} (\mathbf{x}')) \times \mathrm{size}(t)\right] \) containing the approximated solution \( \mathbf{x}(t) \) of the system of ODEs.
