virtual function F(x, x_dot, t)¶

Definition of an implicit system of ODEs of the form:

\[ \mathbf{F}( \mathbf{x}, \mathbf{x}', t ) = \mathbf{0} \]

with optional invariants/hidden constraints of the form:

\[ \mathbf{H}( \mathbf{x}, t ) = \mathbf{0} \]

where \( \mathbf{x} \) are the unknown functions (states) of the independent variable \( t \).

Parameters

  • x – States \( \mathbf{x} \).

  • x_dot – States derivatives \( \mathbf{x}' \).

  • t – Independent variable \( t \).

Returns

The value of the system of ODEs function \( \mathbf{F} \).

virtual function JF(x, x_dot, t)¶

Evaluate the Jacobians with respect to the states \( \mathbf{x} \) and the states defivatives \( \mathbf{x}' \) of the system of ODEs:

\[ \mathbf{JF}_{\mathbf{x}}( \mathbf{x}, \mathbf{x}', t ) = \dfrac{ \partial \mathbf{F}( \mathbf{x}, \mathbf{x}', t ) }{ \partial \mathbf{x} }, \qquad \mathbf{JF}_{\mathbf{x}'}( \mathbf{x}, \mathbf{x}', t ) = \dfrac{ \partial \mathbf{F}( \mathbf{x}, \mathbf{x}', t ) }{ \partial \mathbf{x}' }. \]
Parameters

  • x – States \( \mathbf{x} \).

  • x_dot – States derivatives \( \mathbf{x}' \).

  • t – Independent variable \( t \).

Returns

The Jacobians \( \mathbf{JF}_{\mathbf{x}} \) and \( \mathbf{JF}_{\mathbf{x}'} \) of the ODEs system with respect to the states \( \mathbf{x} \) and the states derivatives \( \mathbf{x}' \).

virtual function JH(x, t)¶

Evaluate the Jacobian of the invariants/hidden contraints of the system of ODEs:

\[ \mathbf{JH}_{\mathbf{x}}( \mathbf{x}, t ) = \dfrac{ \partial \mathbf{H}( \mathbf{x}, t ) }{ \partial \mathbf{x} }. \]
Parameters

  • x – States \( \mathbf{x} \).

  • t – Independent variable \( t \).

Returns

The value of the Jacobian of the invariants/hidden contraints \( \mathbf{JH}_{\mathbf{x}} \).
