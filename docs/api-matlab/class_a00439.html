virtual function Jh_x(x, v, t)¶

Evaluate the Jacobian of the system invariants \( \mathbf{h} \) with respect to the states \( \mathbf{x} \):

\[ \mathbf{Jh}_{\mathbf{x}}( \mathbf{x}, \mathbf{v}, t ) = \dfrac{ \partial \mathbf{h}( \mathbf{x}, \mathbf{v}, t ) }{ \partial \mathbf{x} } + \dfrac{ \partial \mathbf{h} }{ \partial \mathbf{v} } \dfrac{ \partial \mathbf{v}( \mathbf{x}, t ) }{ \partial \mathbf{x} }. \]
Parameters

  • x – States \( \mathbf{x} \).

  • v – Index-1 variables \( \mathbf{v} \).

  • t – Independent variable \( t \).

Returns

The Jacobian \( \mathbf{Jh}_{\mathbf{x}} \).
