class ODEsystem : public indigo.Base¶

Class container for a system of Ordinary Differential Equations (ODEs).

Public Functions

function ODEsystem(name, neqn, ninv)¶

Class constructor for a system of ODEs that requires the following

inputs:

  • name Name of the system of ODEs/DAEs;

  • neqn Number of equations of the system of ODEs/DAEs;

  • ninv Number of invariants/contraints of the system of ODEs/DAEs

    [optional, default = 0].

virtual function F(x, x_dot, t)¶

Definition of the system of ODEs:

\[ \mathbf{F}( \mathbf{x}, \mathbf{x}', t ) = \mathbf{0} \]

with optional invariants of the form:

\[ \mathbf{H}( \mathbf{x}, t ) = \mathbf{0} \]

The derived function must define an implicit system of ODEs of the form \( \mathbf{F}( \mathbf{x}, \mathbf{x}', t ) = \mathbf{0} \) where \( \mathbf{x} \) are the unknown functions of the independent variable \( \mathbf{t} \).

virtual function JF(x, x_dot, t)¶

Evaluate the Jacobian of the of the system of ODEs:

\[ \dfrac{ \partial \mathbf{F}( \mathbf{x}, \mathbf{x}', t ) }{ \hat{\mathbf{x}} } \]

where \( \hat{\mathbf{x}} = \left[ \mathbf{x}, \mathbf{x}' \right]^T \) is the concatenation of the unknown functions and their derivatives.

virtual function H(x, t)¶

Evaluate the invariants of the system of ODEs:

\[ \mathbf{H}( t, \mathbf{x} ) = \mathbf{0}. \]
virtual function JH(x, t)¶

Evaluate the Jacobian of the invariants of the system of ODEs:

\[ \partial \mathbf{H}( t, \mathbf{x} ) / \partial \hat{\mathbf{x}} \]

where \( \hat{\mathbf{x}} = \left[ \mathbf{x}, \mathbf{x}' \right]^T \) is the concatenation of the unknown functions and their derivatives.
