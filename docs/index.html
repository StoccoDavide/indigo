An Integrator for Differential & Differential-Algebraic Equations¶

This MATLAB toolbox is a collection of integrators for ordinary differential and differential-algebraic equations (ODEs/DAEs), which has been originally developed for the course of Mechatronics System Simulation at the University of Trento (Italy) held by Prof. Francesco Biral and Prof. Enrico Bertolazzi.
